What is this?

Tlaloc Sprinklers aims to be an open source, platform agnostic, solution for sprinkler automation and control.

What can I use right now?

Currently tlaloc-adapter-ospi is the only usable part of the system. So your limited command line usage of this gem. If you want to cron your sprinklers, this should get you up and running quickly.

What do I need?

  • Raspberry Pi
  • Open Sprinkler Pi
  • Sprinkler system
